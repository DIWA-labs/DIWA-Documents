# DIWA Licensing Guide

The DIWA project uses a two-tier licensing structure.

## Tier 1 — Engine Core

The DIWA Engine Core contains the underlying engine technologies, including field dynamics, network propagation, clustering systems, memory and hysteresis mechanisms, resistance and elasticity models, runtime update loops, and related engine tooling.

The Engine Core is proprietary and protected under:
• LICENSE_ENGINE_CORE.md

## Tier 2 — Applications, Games, and Research Builds

Applications, games, simulations, research projects, and other software built on top of the DIWA Engine are governed separately.

This tier supports:
• Independent developers
• Educational use
• Noncommercial research
• Games and simulations
• Experimental applications

See:
• LICENSE_APPLICATIONS.md
• RESEARCH_EDUCATIONAL_USE.md

## Commercial Licensing

Certain organizations and use cases require a Commercial License, including:
• Universities conducting funded research
• Government agencies
• Consulting firms
• Organizations exceeding applicable commercial-use thresholds

See:
• COMMERCIAL_LICENSE_POLICY.md

## Contributions

Community contributions are governed by:
• CONTRIBUTOR_AGREEMENT.md

Contributors retain ownership of their Contributions while granting DIWA Research Labs the rights necessary to maintain, distribute, commercialize, and re-license the project.
